1何を問うているか

EEGデコーディングは「時刻tにruleが解読できるか」を問う。jPCAはそれとは別の問いを立てる：

時刻tのpopulation state x(t)から、時刻t+1のstate x(t+1)はどう予測できるか？
→ 表現の「中身」ではなく、表現の「変化の仕方」に着目する

population state とは：N個のニューロン（またはEEGチャンネル）の活動を、あるtimepointにおいてN次元ベクトルとしてまとめたものです。

x(t) ∈ ℝᴺ

例（N=3のとき）:
  t=1: x = [0.2, 0.8, 0.1]  ← ニューロン1,2,3の発火率
  t=2: x = [0.3, 0.6, 0.2]
  t=3: x = [0.5, 0.3, 0.4]
  ...
  → 時間とともにN次元空間を動く「点の軌跡」

2線形ダイナミクスの仮定

最もシンプルな仮定：状態の変化が現在の状態に線形比例する。

ẋ(t) ≈ M x(t)

ẋ = dx/dt（状態の時間微分 = 「次の瞬間どこへ動くか」）
x = N次元の population state
M = [N×N の変換行列]（これを推定したい）

ẋ を有限差分で近似すると：

ẋ(t) ≈ x(t+1) − x(t)  （隣り合うtimepointの差分）
Mを無制約で推定すること自体は簡単（最小二乗法）。問題は、無制約のMは回転・拡大・歪みすべてを含むので、「dynamicsが回転的かどうか」という問いに答えられない。

3Skew-symmetric行列とは

定義

M_skew = −M_skewᵀ   （転置したら符号が逆になる）

図1. Skew-symmetric行列の構造。対角がゼロ、上三角と下三角が符号逆。

なぜ「対称」と「歪対称」に分解できるか

任意の行列 M は必ず次のように一意分解できます：

M =  (M + Mᵀ)/2   +   (M − Mᵀ)/2
        対称成分 S        歪対称成分 A

検証：Aᵀ = (M − Mᵀ)ᵀ/2 = (Mᵀ − M)/2 = −A  ✓
S と A は Frobenius 内積がゼロ（直交）。
→ (M_hat − M_hatᵀ)/2 はM_hatに最も近い（Frobenius距離最小の）skew-symmetric行列。

4なぜ skew-symmetric が「回転」を意味するか

微分方程式の解

ẋ = M_skew x  の解：

x(t) = e^{M_skew · t} x(0)

e^M = Matrix Exponential（行列の指数関数）

Matrix Exponentialが直交行列になる

(e^{M_skew})ᵀ = e^{M_skewᵀ} = e^{−M_skew} = (e^{M_skew})⁻¹

→ e^{M_skew} は直交行列（Oᵀ = O⁻¹）
→ 直交行列 = 回転行列（または鏡映）
→ x(0) が時間とともに回転する（大きさ不変、方向だけ変化）

固有値が純虚数になる理由

M_skew v = λv  とする
両辺に v̄ᵀ を左からかけると：v̄ᵀ M_skew v = λ|v|²

v̄ᵀ M_skew v の複素共役 = −(v̄ᵀ M_skew v)*
→ v̄ᵀ M_skew v は純虚数（または0）
→ λ|v|² が純虚数 → λ = ±iω （ω ∈ ℝ）
固有値 λ = iω のとき、Eulerの公式より e^{iωt} = cos(ωt) + i sin(ωt)。
→ 角周波数 ω でN次元空間内を回転。ω が大きい = 速い回転。

5内積・外積と sin/cos の関係

ベクトルと角度の基本

角度θを持つ方向に長さ1で伸びる単位ベクトルの成分がそのままcosθとsinθです。どんなベクトルも「長さ × 単位ベクトル」に分解できます。

図2. 単位円と角度。sinθ・cosθはベクトルのx,y成分として定義される。

内積がなぜ |a||b|cosθ になるか

2つのベクトルを角度で書いてから内積の定義式を展開すると、三角関数の加法定理によって自然にcosθが出ます。

図3. 内積の導出。「成分ごとの掛け算の和」を展開すると加法定理によりcosθが出る。

外積z成分がなぜ |a||b|sinθ になるか

まったく同じ手順で、a₁b₂ − a₂b₁ を展開すると加法定理の別の形 sin(β−α) が出ます。

a∧b = a₁b₂ − a₂b₁
    = |a|cosα · |b|sinβ − |a|sinα · |b|cosβ
    = |a||b| (cosα sinβ − sinα cosβ)
    = |a||b| sin(β−α)          ← 加法定理（sin の形）
    = |a||b| sinθ
まとめ：「成分の掛け算の和」→ cosθ（内積）、「成分の積を引く」→ sinθ（外積z）。どちらも展開すれば加法定理が出るように作られている。

3Dの外積とは何か、なぜz成分が出てくるか

外積はもともと3Dで定義された操作です。しかし2Dのベクトルは「z成分がゼロの3Dベクトル」と見なせるため、計算するとz成分だけが残ります。

図4. 2Dベクトルを3Dで扱うと、外積のx,y成分はゼロになりz成分だけが残る。これが「2Dの外積」= a₁b₂ − a₂b₁。

6atan2 とは

通常の atan(y/x) は (−π/2, π/2) の範囲しか返せません。(y,x)=(1,1) と (−1,−1) は比が同じなので区別できません。

atan2(y, x) はyとxを別々に受け取り、それぞれの符号から象限を判断して (−π, π] を返します：

y>0, x>0  → 第1象限 → θ ∈ (0, π/2)
y>0, x<0  → 第2象限 → θ ∈ (π/2, π)
y<0, x<0  → 第3象限 → θ ∈ (−π, −π/2)
y<0, x>0  → 第4象限 → θ ∈ (−π/2, 0)

内積が cosθ に、外積z が sinθ に比例しているので：

θ = atan2(sinθ, cosθ)
  = atan2(a∧b / |a||b|,  a·b / |a||b|)
  = atan2(a∧b,  a·b)       ← |a||b| はキャンセル
jPCAでは θ = atan2(x∧ẋ, x·ẋ) として、反時計回り（θ>0）と時計回り（θ<0）を区別します。

7θ による rotation strength の定量化

jPCA平面に射影した後、各（条件c, 時刻t）について：

x(t,c) = jPCA平面上の現在地（位置ベクトル）
ẋ(t,c) = 速度ベクトル（次の瞬間どこへ動くか）
θ(t,c) = atan2( x∧ẋ,  x·ẋ )

図5. θの4つのケース。θ = π/2 が pure rotation（速度が軌跡に垂直）、θ = 0が pure expansion（速度が位置と平行）。

θ分布の読み方（Churchland 2012 Fig.6a）

ピーク ≈ π/2  → rotational dynamics あり → jPCAが有効
ピーク ≈ 0°   → expansion/contractionが支配的
フラットな分布 → 構造なし（ランダムノイズ）
Churchland 2012では実データとgenerator modelのθ分布がともにπ/2にピーク。velocity-tuned modelやEMGはピークが0付近（rotation説の否定例として使用）。

8R² 比による rotation strength の定量化

θとは独立した別の指標。

R²_skew    = M_skew で dX をどれだけ説明できるか
R²_unrestr = 制約なしMでどれだけdXを説明できるか（上限）

R²_ratio = R²_skew / R²_unrestr
1.0 に近い → dynamicsがほぼ純粋な回転で説明できる（skew-symmetric制約を課してもほとんど損しない）
0.5 前後 → ランダムデータの期待値（M_hatの対称成分と歪対称成分が同程度の説明力を持つため）
0.5 を大きく下回ることはない（M_skewはM_hatの部分集合）

9アルゴリズムのまとめ

Input は条件ごとの [N × T] 行列のリスト。N = features（EEGチャンネルなど）、T = timepoints、条件数 ≥ 3 が必要（binary conditionでは回転平面が1次元になり無意味）。

Step 1：PCA前処理

(a) 全条件を結合
    X_full [N × C*T]  ← C条件を横に並べる

(b) cross-condition mean を引く
    各timepointについて、全条件の平均を引く
    → X_dm = X_full − cross_mean_broadcast
なぜcross-condition meanを引くか？
全条件に共通するevoked response（例：刺激後に全条件で一斉に上がる成分）があると、jPCAはそのドリフトを「回転」として誤検出する。条件間の差分だけを残すことで、純粋な構造を見る。 
(c) PCA → top n_pcs PCs を保持（典型的に n_pcs = 6）
    pca = prcomp(t(X_dm))
    X_red [n_pcs × C*T]  ← 上位6PCに削減
なぜPCAが必要か？
N次元全体でM_skewを推定すると過学習する。分散の大きい成分だけに絞ることで、ノイズに乗ったrotationを拾わない。

Step 2：有限差分で ẋ を近似

各条件について：
  dX[t]    = X[t+1] − X[t]   ← 隣り合うtimepointの差分
  X_prev   = X[1:(T-1)]       ← 最後のtimepointは除外

条件をまたいで結合：
  dX     [n_pcs × C*(T-1)]
  X_prev [n_pcs × C*(T-1)]
各条件の最後のtimepointを除くのは、「条件Aの最後」と「条件Bの最初」が差分計算で混ざるのを防ぐため。

Step 3：無制約 M を最小二乗推定

dX ≈ M X_prev

→ M_hat = dX X_prevᵀ (X_prev X_prevᵀ)⁻¹
導出：
dX = M X_prev の両辺に右から X_prevᵀ (X_prev X_prevᵀ)⁻¹ をかけるだけ。最小二乗の標準的な解。

Step 4：Skew-symmetric に射影

M_skew = (M_hat − M_hatᵀ) / 2
なぜこれが最も近いskew-symmetric行列か？
任意の行列は対称成分Sと歪対称成分Aに一意分解できる（Section 3参照）。(M_hat − M_hatᵀ)/2 はM_hatの歪対称成分Aだけを取り出す操作。SとAはFrobenius内積がゼロ（直交）なので、Aを取り出してもM_hatからの距離は最小になる。

Step 5：M_skew の固有値分解

eigen(M_skew)
  → eigenvalues:  ±iω₁, ±iω₂, ...  （純虚数のペア）
  → eigenvectors: 複素共役ペア (v₁, v̄₁), (v₂, v̄₂), ...

|ω| の大きい順に並べ替え
  → ω が大きい = 速い回転 = 強い rotational dynamics
固有値が純虚数なのはskew-symmetricの性質（Section 4参照）。固有ベクトルが複素数なのも同じ理由で、次のステップで実数に戻す。

Step 6：実数の回転平面を復元

複素共役ペア (v₁, v̄₁) から実数の平面基底を作る：

jPC1 = 2 Re(v₁)     ← 実部の2倍
jPC2 = −2 Im(v₁)    ← 虚部の負の2倍

なぜこの式か：
  v₁  = Re(v₁) + i Im(v₁)
  v̄₁  = Re(v₁) − i Im(v₁)
  v₁ + v̄₁  = 2 Re(v₁)        → jPC1
  i(v₁ − v̄₁) = −2 Im(v₁)    → jPC2
  この2つは直交し、複素固有空間と同じ部分空間を張る

各jPCを単位ベクトルに正規化。
net rotationが反時計回りになるよう方向を揃える。

Step 7：データを射影

W = [jPC1; jPC2]   [2 × n_pcs]

X_jPCA = W X_red   [2 × C*T]

→ 条件ごとに分割すると各条件の2D軌跡が得られる

Output

W          : jPC axes（回転平面を定義する2本の軸）  [2 × n_pcs]
X_jPCA     : 2D projection（各条件・各timepointの座標）  [2 × C*T]
M_skew     : fitted dynamics matrix  [n_pcs × n_pcs]
M_unrestr  : unconstrained M（比較用）
R²_ratio   : rotation strength（M_skew / M_unrestr）
θ分布      : rotation strength（角度ベース、Section 7参照）

10Comprehension Check

実装前にすべて答えられること：

  1. M_skewの固有値がなぜ必ず純虚数になるのか？ ヒント：skew-symmetricの定義から、v̄ᵀ M_skew v がどんな種類の数になるかを示す
  2. jPC1とjPC2はなぜ必ず直交するのか？ ヒント：2Re(v₁) と −2Im(v₁) の内積を計算してみる
  3. cross-condition meanを引かないと何が起きるか？ ヒント：全条件に共通する強いevoked responseがあったとき、M_hatはそのドリフトをどう「説明」しようとするか
  4. R²_ratio = 0.5はなぜランダムデータの期待値か？ ヒント：ランダムなM_hatに対して、対称成分Sと歪対称成分Aの説明力はどう配分されるか
  5. x∧ẋ の符号が負のとき、回転はどちら向きか？ ヒント：jPCAが「反時計回りを正」とするとき、θ < 0 はどう解釈するか
  6. なぜ条件数が3以上必要か？ ヒント：binary conditionのとき、jPCA平面への射影は何次元になるか
