jPCA — 幾何的補足

このページは jPCA_notes.md の Section 4（角度 θ による回転強度）の 視覚的補足です。テキストだけではイメージしにくい jPC 平面における velocity の幾何 を図で示します。

1. jPC 平面とは

jPCA は N 次元の population state を、最も回転的な 2 次元平面（jPC1–jPC2 平面）に射影します。

各条件の軌跡が 反時計回り を描き、velocity ベクトル（矢印）が常に軌跡の接線方向を向いているとき、angle θ ≈ π/2 となります。

2. θ の 4 ケース

速度 ẋ が位置 x と垂直（反時計回り）
x · ẋ = 0, x ∧ ẋ > 0

速度が位置と同方向（放射状に拡大）
x · ẋ > 0, x ∧ ẋ = 0

速度が位置と垂直（時計回り）
x · ẋ = 0, x ∧ ẋ < 0

速度が位置と逆方向（原点に収束）
x · ẋ < 0, x ∧ ẋ = 0

3. θ 分布の読み方

すべての (条件 c, 時刻 t) について θ を計算し、ヒストグラムを作ります。

θ 分布の形解釈
π/2 にシャープなピーク強い rotational dynamics（jPCA が有効）
0 付近にピークexpansion/contraction が支配的
フラット構造なし（ランダムノイズ）

Churchland 2012 Fig.6a では、運動野の実データと generator model がともに π/2 にピークを持ち、velocity-tuned model や EMG データはピークが 0 付近にあることが示されている。

4. ピーク角度のバイアスについて

実装上の注意（jPCA_lib.R での観察）：
純粋な回転データ（omega = 0.2）で jpca_rotation_strength() を実行すると、 ピーク角度は理論値 π/2 ≈ 1.571 よりも大きく（≈ 1.775）なる。これは 2 つの系統的バイアスに由来する：

  1. 有限差分バイアス：連続時間の円運動 θ(t)=ωt を有限差分で近似すると、 1 timepoint 分の時間遅れ（= ω/2 rad の付加位相）で速度が計算される。 omega=0.2 のとき +0.1 rad 程度のバイアス。
  2. 正規化バイアス：PC ごとの軸スケールが違うと jPC 平面が楕円状に歪み、 atan2 で計算される角度がシフトする。normalize=TRUE を使っても完全には解消されない。

このバイアスは系統的なものであり、ノイズとは無関係。閾値 |peak − π/2| < 0.3 でのテスト合格は意図的な許容範囲。
