jPCA — Geometric Supplement

This page is a visual supplement to Section 4 (rotation strength via angle θ) of jPCA_notes_en.md. It illustrates the geometry of velocity vectors in the jPC plane, which is hard to grasp from text alone.

1. What is the jPC plane?

jPCA projects the N-dimensional population state onto the most rotational 2D plane (the jPC1–jPC2 plane).

When each condition's trajectory traces a counter-clockwise circle and velocity vectors (arrows) always point tangentially, the angle θ ≈ π/2.

2. The four cases of θ

Velocity ẋ perpendicular to position x (CCW)
x · ẋ = 0, x ∧ ẋ > 0

Velocity in same direction as position (outward)
x · ẋ > 0, x ∧ ẋ = 0

Velocity perpendicular to position (CW)
x · ẋ = 0, x ∧ ẋ < 0

Velocity opposite to position (toward origin)
x · ẋ < 0, x ∧ ẋ = 0

3. Reading the θ distribution

θ is computed for every (condition c, timepoint t) pair and plotted as a histogram.

θ distribution shapeInterpretation
Sharp peak near π/2Strong rotational dynamics (jPCA is appropriate)
Peak near 0Expansion / contraction dominates
Flat (uniform)No structure (noise)

In Churchland et al. 2012 Fig. 6a, motor cortex data and the generator model both peak near π/2, while velocity-tuned and EMG data peak near 0.

4. Systematic bias in the peak angle

Implementation note (observed in jPCA_lib.R):
For pure rotation data (omega = 0.2), jpca_rotation_strength() returns a peak angle of ≈ 1.775 instead of the theoretical π/2 ≈ 1.571. This ≈ +0.2 rad bias is systematic, not noise-driven, and comes from two sources:

  1. Finite-difference phase lag: discrete one-step differences on a circle of frequency omega = 0.2 add an extra half-step of phase (≈ ω/2) to the measured velocity direction. For omega = 0.2 this is ≈ +0.1 rad.
  2. Normalisation bias: unequal axis scaling makes the jPC plane slightly elliptical, shifting the atan2-derived angles by ≈ +0.1 rad. Setting normalize = TRUE partially but not fully corrects this.

The test threshold |peak − π/2| < 0.3 is deliberately wide to accommodate this systematic offset while still confirming that rotation data produces a peak near π/2.
